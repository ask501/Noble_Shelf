Features
すべての本を、一箇所に。
電子書籍・同人誌・自炊本のローカル管理に特化した、軽量で高機能なライブラリツール。
ドラッグ&ドロップで登録
ZIP・フォルダ・PDFをドロップするだけで即登録。複数同時投入にも対応し、重複は自動検出。
メタデータ自動取得
購入した作品の情報をブックマークレット経由で一発取得。タイトル・サークル・タグを自動補完。
評価・お気に入り
星評価やお気に入り登録でコレクションを整理。グリッドと連携してすぐに絞り込める。
高速サムネイルグリッド
仮想描画による超高速グリッド表示。数千冊のコレクションでもスムーズにスクロール。
ビルトインビューア
JPG・PNG・WEBP・GIF・BMPに対応したビューアを内蔵。外部アプリ不要で、ライブラリからそのまま閲覧できる。
ビューア連携機能
ストア購入ファイルも、そのまま起動。外部アプリとの連携で、どんな形式もアプリから直接開ける。
ライブラリ管理
タイトル・サークル・タグで瞬時に検索。豊富なフィルターでコレクションを自在に整理できる。
自動バックアップ
ライブラリ変更時に自動でDBをバックアップ。タイムスタンプ付きで世代管理。
