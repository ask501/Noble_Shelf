# プラグイン（メタデータ取得）

このフォルダがプラグインの配置先です。メニュー「プラグイン」→「プラグインフォルダを開く」で開けます。

**読み込みルール**

- **スキャン対象**: `plugins/` 直下、または `plugins/get_api/` が存在する場合はその直下。
- いずれかの直下にある**サブフォルダ**のうち、`__init__.py` を持つものが1つ1つプラグインとして読み込まれます（`_` で始まる名前のフォルダは除外）。

**契約**

各プラグインの `__init__.py` で、次の属性を持つオブジェクトを export します（モジュール直下に定義するか、`get_api()` で返す）。

- 必須: `PLUGIN_NAME`, `PLUGIN_SOURCE_KEY`, `search_sync`, `get_metadata_sync`
- 任意: `can_handle(product_id_or_url)`, `get_property_buttons(context)`（プロパティ／ストア追加ダイアログに並べるボタン）

本体の `plugin_loader` が上記を満たすかチェックし、満たすものだけ有効なプラグインとして扱います。メニュー「プラグイン」で各プラグインの ON/OFF を切り替えられます。
